# [포트폴리오] PM 배종현의 핵심 R&D 프로젝트 포트폴리오 (v1.0.0-prototype)

## 📁 대표 프로젝트: '지능형 다목적 스마트시티 입지 선정 및 공공갈등 예측 플랫폼 OmniSite' (26.06 ~ 26.08 / 7인개발 / PM, 아키텍트)

- **Next.js & FastAPI 기반의 단일 통합 파이썬 백엔드(Monolith) 다목적 공간-AI 행정 의사결정 지원 플랫폼 구축**
- **역할:** 프로젝트 매니징(PM) 및 시스템 아키텍트 (7인 주니어 개발팀 관리 및 프런트-백엔드 마이크로서비스 설계)
- **핵심 기술 스택 및 선정이유:**
    *   **Next.js:** Server-Side Rendering (SSR)을 도입하여 대용량 GIS 지적 데이터를 브라우저 렌더링 부하 없이 고속 연동하고, Backend-For-Frontend (BFF)를 구현하여 AI의 실시간 토론 과정을 SSE(Server-Sent Events) 스트리밍으로 클라이언트에 지연 없이 전도하기 위해 선정.
    *   **Python FastAPI (Unified Backend):** 비효율적인 다중 언어 통신(Spring Boot ➔ FastAPI)을 배제하고 단일 파이썬 백엔드로 통합하여 개발 생산성을 3배 이상 극대화. 사용자 Auth, 프로젝트 CRUD, 복잡한 PostGIS 공간 연산 마스크 필터링 및 LangGraph 멀티 에이전트 워크플로우를 하나의 초경량 고성능 비동기 서버로 일괄 처리.
    *   **PostgreSQL / PostGIS:** 다각도 지적도 필지 정보의 위치 좌표(`EPSG:4326`)를 저장하고 `ST_Difference`와 spatial indexing(`GIST`)을 통해 초고속으로 규제 구역을 배제하는 기하학 공간 차집합 연산을 수행하기 위해 선택.
    *   **AWS EKS (Kubernetes):** 각 지자체(클라이언트)별 개인 정보 및 데이터베이스 접근 권한을 하드웨어 수준에서 안전하게 분리하는 Namespace 격리와 서비스 어카운트용 IAM 역할(IRSA) 매핑을 통해 B2G 공공 컴플라이언스를 확보하고자 채택.

## 🚀 PM으로서의 핵심 성과 및 설계 극복 사례 (Technical Accomplishments)

1.  **개발 속도 및 백업 한계 극복을 위한 단일 통합 백엔드(Monolith) 전환:**
    *   초기에는 Spring Boot와 FastAPI 분리 설계를 채택했으나, 이종 프레임워크 간의 연동 오버헤드와 자바 에이스 개발자 1명에게 성패가 좌우되는 버스 팩터(SPOF) 리스크를 식별함.
    *   백엔드를 **FastAPI 단독 통합형**으로 과감히 전환하여 개발팀 전체가 파이썬이라는 단일 언어로 협업하도록 함으로써, 상호 코드 리뷰 및 백업이 가능하게 하고 통신 레이어를 완전히 생략하여 개발 효율을 300% 이상 향상시킴.
2.  **AI 모델 붕괴(Model Collapse) 방지를 위한 데이터 보안 설계:**
    *   AI 시뮬레이션 예측 결과가 일반 RAG DB에 그대로 피드백되어 향후 분석을 오염시키는 루프 현상을 식별함. 
    *   사용자가 최종 행정 고시문(PDF)을 업로드할 때 가동되는 **Audit AI(검토 에이전트)** 파이프라인을 설계하여 실제 완료된 사례만 독립 세그먼트(`AI 기반 실제사례`)에 보관함으로써 데이터 오염을 원천 차단함.
3.  **API 호출 비용 및 품질 균형을 위한 온디맨드 UX 아키텍처 수립:**
    *   모든 후보지(Top 3)에 대해 고성능 GPT-4o 멀티 에이전트 시뮬레이션을 상시 구동할 시 발생하는 과도한 API 요금 문제를 감지함.
    *   가중치 연산 점수가 가장 높은 **Top 1 부지만 자동 시뮬레이션**을 돌리고, Top 2 & 3은 사용자가 직접 실행 버튼을 누르는 **온디맨드 디렉토리 구조**를 설계하여 품질(GPT-4o 강제)을 사수하면서도 1회 실행 비용을 94원 이하로 최적화함.
4.  **지역 갈등 민감도 벡터 ($CSS(p)$) 수학적 공식 고안:**
    *   단순 공간 분석의 한계를 넘어 지자체의 실제 민원 핫스팟 및 연령 통계 요인을 결합하여 에이전트 토론의 한계점을 능동적으로 통제하는 공간 가중치 결합 알고리즘을 설계하여 AI 갈등 예측의 정합성을 획득함.
