# [설계서] OmniSite 플랫폼 최종 UI/UX 설계 명세서 (v1.0.0-prototype)

본 설계서는 KT 에이블스쿨(AIVLE) UI 설계서 표준 양식에 맞춰 작성되었으며, 조원들이 메인 대시보드(OMS-01-01-001)의 프리미엄 다크 모드 및 글래스모피즘 테마를 플랫폼 전체 화면에 일관되게 이식할 수 있도록 정의한 공식 UI/UX 디자인 가이드라인입니다.

> [!NOTE]
> **디자인 톤앤매너 안내**
> 메인 화면의 프리미엄 다크 네이비 테마(`hsl(222, 47%, 6%)`)와 형광 블루/적색/녹색 악센트, `backdrop-filter: blur(12px)` 글래스모피즘 레이어 스타일이 4개 핵심 화면 전체에 100% 동일하게 적용됩니다. 
> *(※ 현재 AI 이미지 생성 쿼터가 일시 소진되어, 추가 Mockup 이미지 생성은 소진 리셋 이후 적용할 예정이며, 본 문서에는 프런트엔드 개발자가 즉각 참고할 수 있는 정밀 레이아웃 코드 및 마크다운 와이어프레임을 우선적으로 탑재했습니다.)*

## 🎨 1. 전역 디자인 시스템 및 스타일 가이드 (CSS Tokens)

모든 UI 컴포넌트는 아래 전역 CSS 토큰을 공유하여 퍼블리싱해야 합니다.

```css
:root {
  /* HSL 프리미엄 다크 컬러 시스템 */
  --bg-main: hsl(222, 47%, 6%);         /* 전체 메인 배경 */
  --bg-surface: hsla(223, 47%, 11%, 0.75); /* 반투명 유리 패널 배경 */
  --border-glass: rgba(255, 255, 255, 0.08); /* 패널 경계선 */
  
  /* 상태 및 강조 색상 */
  --color-accent: hsl(217, 91%, 60%);   /* 하이라이트 블루 */
  --color-danger: hsla(354, 84%, 57%, 0.4);  /* 배제 마스크/경고 (Red) */
  --color-success: hsla(142, 70%, 50%, 0.65); /* 후보지/안정 (Green) */
  --color-text-primary: hsl(0, 0%, 100%);  /* 흰색 본문 */
  --color-text-secondary: hsl(215, 20%, 65%); /* 회색 서브텍스트 */
}

/* 글래스모피즘 패널 기본 클래스 */
.glass-panel {
  background: var(--bg-surface);
  backdrop-filter: blur(12px);
  border: 1px solid var(--border-glass);
  border-radius: 16px;
  box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.5);
  color: var(--color-text-primary);
  font-family: 'Inter', sans-serif;
}
```
