## 🏁 2. 조장(테크 리더)의 개발 통제 및 코드 리뷰 3대 철칙 (Code Review Rules)
1.  **DDL 검수 시 GIST 인덱스 누락 즉시 반려:** 주니어 C가 지오메트리 테이블 생성 시 GIST 인덱스 선언을 빠뜨린 PR은 즉시 Closed(반려)할 것.
2.  **FastAPI async def 미적용 시 즉시 반려:** 주니어 D가 백엔드 API 라우터 함수를 일반 `def`로 정의한 경우, 동기 블로킹이 발생하므로 무조건 `async def` 및 `await httpx.AsyncClient` 호출로 리팩토링 지시할 것.
3.  **AHP Lock 미구현 및 C.R. 예외 분리 누락 시 즉시 반려:** 주니어 B/D가 AHP 연산 화면에서 C.R. 수치 제한 없이 저장이 가능하도록 구현한 커밋은 병합(Merge)을 불허할 것.
