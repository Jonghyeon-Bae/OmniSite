# [v1.2-alpha] AI 모의 토론 멀티에이전트화 및 관리자 콘솔 UI/UX 고도화 계획서

본 계획서는 **OmniSite v1.2-alpha**의 고도화된 기능 완공을 위해 다음의 4대 핵심 보완 사안을 물리/논리적으로 설계하고 적용하기 위한 구현 지침서입니다.

---

## User Review Required

> [!IMPORTANT]
> 1. **AI 페르소나 그룹 멀티에이전트(Multi-agent) 토론 루프 전환:**
>    * 기존의 단일 LLM 세션 대본 생성(Single-session Prompting) 방식을 폐지합니다.
>    * **찬성측(상인/데이터 전문가)**, **반대측(주민/생활 체감)**, **정부측(조정관/중재 관료)**의 3대 독립 에이전트 세션을 구축합니다.
>    * 대화 스레드 역사(`chat_history`)를 공유하며 각 에이전트 LLM이 이전 발언을 수신하여 턴(Turn) 단위로 연쇄 응답을 생성하는 **독립 에이전트 교차 토론 루프**를 백엔드에 구축합니다.
>    * 이로써 한 화자 안에 여러 대사가 뒤엉키는 **화자 뭉침 현상**을 원천 해결하며, 화자별 줄바꿈(`\n\n`)과 콜론(`:`) 분할 규격을 명확히 보장합니다.
> 2. **벌크 적재 ⓘ 도움말 툴팁 인포 단추 신설:**
>    * 데이터 벌크 탭에서 실무자가 어떤 테이블에 어떤 포맷(CSV/SHP 필수 컬럼 스펙 등)을 적재해야 하는지 즉각 인지하도록 마우스 호버형 가상 툴팁(`i` 아이콘)을 제공합니다.
> 3. **통합 관리자 콘솔 모달 X축 확장:**
>    * 관리자 콘솔 모달 너비를 기존 `max-w-lg`에서 `max-w-4xl`(또는 `max-w-3xl`)로 300~400px 대폭 확장하여, 데이터 테이블 및 ML 피처 막대 그래프의 시각 가독성을 확보합니다.
> 4. **이쁜 커스텀 토스트 알림(Toast Notification) 도입:**
>    * 비밀번호 변경, API 적재 성공/실패 시 브라우저 기본 경고창(`alert()`)을 띄우던 방식을 지양하고, 시스템 어두운 유리모피즘 테마에 부합하는 **이쁜 커스텀 토스트 팝업**을 구현해 시각 통일성을 부여합니다.

## Proposed Changes

### 1. 백엔드 (FastAPI) 멀티에이전트 토론 파이프라인

#### [MODIFY] [spatial.py](file:///c:/Users/Admin/Desktop/빅프로젝트 관련자료/최종1차/1.0-prototype/backend/app/routers/spatial.py)
* **`POST /api/v1/spatial/debate` (멀티에이전트 토론 스트리밍):**
  * 3개 역할(찬성, 반대, 정부)에 대한 **개별 System Prompt**를 독립적으로 선언합니다.
    * **상인대표 (Merchant Agent):** 정량 수치(유동인구), AHP 가중치 기여율을 인용해 경제적 당위성을 주장.
    * **주민대표 (Resident Agent):** 민원 수, 무단투기 실태를 인용해 주민 주거권, 정주 환경 침해를 감정적/아날로그적 억양으로 성토.
    * **갈등조정관 (Coordinator Agent):** pgvector RAG 조례 법리를 들이대며 중립적 입장에서 완충 이격거리 확대 및 삼진아웃 가동정지권 위임 등의 타협안 제안.
  * `chat_history = []` 리스트를 선언하여 대화 내역을 유지합니다.
  * 총 8턴의 루프를 돌며 각 턴에 지정된 에이전트 LLM을 호출하여 생성된 대화 텍스트를 스트리밍 형식으로 전송하고, 생성 완료 시 `chat_history`에 추가해 다음 에이전트의 입력 문맥으로 공급합니다.
  * API 키 부재 시 실행되는 **Mock Fallback** 역시 턴마다 일정한 시차를 두고 개별 스트리밍을 yield 하도록 재구성하여 멀티에이전트처럼 연출합니다.

### 2. 프론트엔드 (Next.js) 어드민 UI/UX 개조

#### [MODIFY] [page.js](file:///c:/Users/Admin/Desktop/빅프로젝트 관련자료/최종1차/1.0-prototype/frontend/src/app/spatial/page.js)
* **커스텀 토스트 알림 상태 추가 및 헬퍼 함수 구현:**
  * `const [toast, setToast] = useState({ show: false, message: '', type: 'info' })` 선언.
  * `showToast(msg, type)` 함수를 통해 자동으로 3초 후 페이드아웃되는 경량 토스트 뷰어를 모달 내부 또는 화면 상단 중앙에 렌더링합니다.
  * 기존 `alert()` 호출부들을 `showToast()`로 전량 대체하여 행정 알림 디자인을 개선합니다.
* **통합 관리자 콘솔 모달 크기 확장:**
  * 모달의 CSS 레이아웃을 `max-w-lg` (512px) ➔ `max-w-3xl` (768px) 또는 `max-w-4xl` (896px)로 확장 적용합니다.
* **원천 데이터 벌크 적재 인포 단추 및 호버 툴팁 배치:**
  * 데이터 벌크 탭 내 `🚀 원천 데이터 벌크 적재` 타이틀 우측에 `ⓘ` 아이콘을 배치하고, 마우스 호버(`group-hover`) 시 테이블별 CSV/Shapefile 필수 스키마 사양을 리스트업하는 툴팁을 오버레이 표시합니다.
