# 대시보드 HTML 리포트 동적 실측 연동 및 코드 최적화 완료 보고서 (v1.2.0-alpha-Rev91)

대시보드 HTML 행정 보고서 발급 기능 내에 남아있던 필지 데이터 및 예산 수치의 하드코딩 요소를 제거하고, 실제 DB에서 쿼리된 지적 면적 및 공시지가를 기반으로 연간 도로점용료가 실시간으로 동적 산출되도록 보완을 완료했습니다.

## 🛠️ 주요 변경 및 기술적 성과

1. **대시보드 HTML 리포트 동적 실측 연동 (`dashboard/page.js`)**
   - 기존의 고정 지적 면적 `15.0 ㎡` ➔ 실제 선정된 필지의 면적(`${item.selectedParcelArea} ㎡`)으로 연동.
   - 기존의 고정 공시지가 `₩ 14,200,000 / ㎡` ➔ 실제 선정 필지의 지가(`${item.selectedParcelPrice}`)로 연동.
   - **도로점용료 실시간 계산**: 지적 면적과 선정지 기준 공시지가 및 법정 요율 2%를 실시간 연산하도록 템플릿 개조 완공 (`selectedParcelArea * selectedParcelPrice * 0.02`).

2. **미사용 목업 코드 완전 소멸 정리**
   - 과거 프론트엔드에 하드코딩되어 있던 모의 토론 더미 목업 함수 `getMockDebateLogs` 를 안전하게 완전 폐기하였으며, 실제 PostgreSQL 데이터베이스(`decision_histories` 테이블)에 적재된 실제 의사결정 이력 데이터만을 표출하도록 100% 정상화하였습니다.
